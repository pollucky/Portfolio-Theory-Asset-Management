This repository contains Python implementations of key concepts from the Portfolio Theory & Asset Management course, part of the Master’s program “Financial Markets and Institutions” at Faculty of Economics, Lomonosov Moscow State University.

The project uses real data from the MOEX ISS API and includes:

- stock price and dividend data integration
- analysis of returns (price vs. total return)
- risk metrics (volatility, VaR, Expected Shortfall)
- distribution analysis and normality testing
- construction and comparison of portfolios

**Objective**: To apply portfolio theory to real market data and analyze the effects of dividends, risk, and diversification.
